## 🎯 המיצוב שלנו — מי אנחנו ולמי אנחנו מדברים

### המותג:
**Thailand Moments** = קהילת ישראלים שחיים בתאילנד / עברו אליה / מתכננים לעבור.

### ההבדל מ-Thailand Vibes:
| Thailand Vibes | Thailand Moments |
|---|---|
| סוכנות נסיעות מסחרית | קהילה אותנטית של מקומיים |
| תיירים שמגיעים לחופשה | ישראלים שחיים / עוברים לגור |
| "בנו לכם את הטיול" | "אנחנו כבר שם — בואו אלינו" |
| שירותי הזמנה (מלונות/טיולים) | מדריכי חיים, ויזה, שכירות, קהילה |
| מכירה | השראה + מידע + שייכות |

### הקהל שלנו — 3 פרסונות:
1. **"המתכנן"** — ישראלי שחולם על רילוקיישן, בשלב מחקר ראשוני
2. **"המחליט"** — כבר החליט לעבור, צריך מידע פרקטי (ויזה, דירה, כסף)
3. **"המקומי"** — כבר גר בתאילנד, מחפש קהילה וטיפים

## 📝 אסטרטגיית פוסטים — Template לכל פוסט

### המבנה האידיאלי של פוסט:

```
1. כותרת SEO חזקה (H1)
   דוגמה: "צ'יאנג מאי — המדריך המלא לישראלים שחיים שם"

2. תמונה ראשית (Featured Image) — איכות גבוהה, רוחבית

3. פסקת פתיחה (Hook) — 2-3 משפטים שגורמים להמשיך לקרוא
   - שאלה / עובדה מפתיעה / אנקדוטה אישית

4. תוכן עניינים (Table of Contents) — לפוסטים ארוכים

5. הגוף — חלקים עם H2/H3 ברורים:
   ✅ מה לדעת לפני שמגיעים
   ✅ איפה לגור / שכונות
   ✅ מה לעשות / לראות
   ✅ איפה לאכול
   ✅ כמה עולה
   ✅ איך מגיעים
   ✅ טיפים מקומיים (הזווית שלנו!)

6. CTA באמצע הפוסט — "הצטרף לקהילה"

7. סיכום + CTA בסוף — "הורד את המדריך"

8. Internal Links — 3-5 קישורים לפוסטים קשורים
```

## 🚀 סדר העדיפויות — מה לכתוב ראשון

### שלב א׳ — השבועות הראשונים (High Impact):
1. **"כמה עולה לחיות בתאילנד"** — ביטוי חיפוש חם מאוד
2. **"מציאת דירה בתאילנד לישראלים"** — כאב ספציפי
3. **"בנגקוק — איפה כדאי לגור?"** — תנועה גבוהה
4. **שיפור פוסטים קיימים** — SEO + CTA + Internal Links

### שלב ב׳ — חודש שני:
5. **"ביטוח בריאות לגרים בתאילנד"**
6. **"העברת כסף מישראל לתאילנד"**
7. **"פוקט — האם שווה לגור שם?"**
8. **"המסעדות הכי טובות בצ'יאנג מאי"**

### שלב ג׳ — חודש שלישי:
9. תוכן מתקדם על ויזה (חידושים, ניסיונות אישיים)
10. השוואות ערים (צ'יאנג מאי vs בנגקוק)
11. מדריכי חיים נישתיים (ילדים, כלבים, גיל פרישה)

## 🔗 מערכת Internal Linking

### כל פוסט חייב לקשר ל:
1. **עמוד הקהילה** (whatsapp) — תמיד
2. **דף המדריך** (lead magnet) — תמיד
3. **עמוד היעד הרלוונטי** (צ'יאנג מאי / בנגקוק וכו')
4. **2 פוסטים קשורים** — מאותה קטגוריה

### Pillar Pages (עמודי עמוד):
כל עיר/יעד צריך להיות עמוד ראשי שמקשר לכל הפוסטים שלו:
- צ'יאנג מאי → כל הפוסטים על צ'יאנג מאי
- בנגקוק → כל הפוסטים על בנגקוק
- וכו'
